基于vuejs全家桶搭建客户端渲染单页面。

## Node 版本

```bash
node 8.x +
```

## 安装国内镜像
```bash
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
> 安装完国内镜像后npm命令都改为cnpm。

## 终端工具
> [http://cmder.net/](http://cmder.net/)windows的同学可以安装下这个cmd第三方工具，可以支持linux 命令


## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
npm run dev
```
### Test
```bush
npm run debug
```
### Production(Build)
```bush
npm run build
```
